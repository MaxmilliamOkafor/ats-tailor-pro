# Icon Placeholder Instructions

To complete your Job Genie installation, you need to add three icon files to this directory:

## Required Icons:

1. **icon16.png** - 16x16 pixels
2. **icon48.png** - 48x48 pixels  
3. **icon128.png** - 128x128 pixels

## Option 1: Quick Solution (Recommended)

Use any PNG image you have (can be the same image for all three):
- Resize to 16x16, 48x48, and 128x128 pixels
- Name them accordingly
- Place them in this folder

- **16x16**: Shows in browser toolbar
- **48x48**: Shows in extension management page
- **128x128**: Shows in Chrome Web Store (if publishing)
